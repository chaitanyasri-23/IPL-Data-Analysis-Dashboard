# IPL Data Analysis Dashboard

##  Project Overview

This project is an interactive **IPL Data Analysis Dashboard** built using **Power BI**. It provides insights into IPL matches, teams, players, toss decisions, and overall tournament statistics through interactive visualizations.

---

##  Tools Used

- Power BI
- Power Query
- DAX (Basic)
- IPL Dataset

---

##  Dashboard Features

-  Total Matches
-  Total Runs
-  Total Balls
-  Total Wickets
-  Top Winning Teams Analysis
-  Toss Decision Distribution
-  Season-wise Match Trend
-  Top Players of the Match

---

##  Key Insights

- Identified the teams with the highest number of wins.
- Analyzed toss decision patterns.
- Visualized season-wise match trends.
- Highlighted the top "Player of the Match" award winners.
- Summarized important IPL statistics using KPI cards.
